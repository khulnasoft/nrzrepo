title: Vercel
description: Learn how to use Nrzrepo on Vercel.
---

![](/images/docs/nrzrepo-x-vercel.png)

Vercel's zero-config integration with Nrzrepo automatically understands your monorepo.

To deploy your Nrzrepo on Vercel, [create a new project](https://vercel.com/new) and import your code. Your projects will be pre-configured with the correct settings to use the [Vercel Remote Cache](https://vercel.com/docs/monorepos/remote-caching).

For more information about deploying your Nrzrepo to Vercel, [visit the Vercel documentation](https://vercel.com/docs/concepts/monorepos/nrzrepo).
