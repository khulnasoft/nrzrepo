---
title: login
description: API reference for the `nrz login` command
---

Log in to your Remote Cache provider.

The default provider is [Vercel](https://vercel.com/). To specify a different provider, use the `--api` option.

## Flag options

### --api \<url>

Set the API URL of the Remote Cache provider.

```bash title="Terminal"
nrz login --api=https://acme.com/api
```

### --login \<url>

Set the URL for login requests. This is only required for platforms with robust login features.

```bash title="Terminal"
nrz login --login=https://acme.com
```

### --sso-team \<team>

Connect to an SSO-enabled team by providing your team slug.

```bash title="Terminal"
nrz login --sso-team=slug-for-team
```
